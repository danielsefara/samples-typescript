# Food Delivery App

- Worker code: [`src/worker.ts`](./src/worker.ts)
- Workflow build script: [`src/scripts/build-workflow-bundle.ts`](./src/scripts/build-workflow-bundle.ts)

### Running this sample

1. Make sure Temporal Server is running locally (see the [quick install guide](https://docs.temporal.io/server/quick-install/)).
1. `npm install` to install dependencies.
1. `npm run dev` to start the Next.js dev servers and the Temporal Worker.
1. Interact with the sites:

- Menu: http://localhost:3000
- Driver portal: http://localhost:3001
